# BetterInfo+

Personal fork of <cy>BetterInfo</c> by <cl>Cvolton</c>, with extra level browser tools. <cr>Do not run this alongside the original BetterInfo mod</c> — it replaces it entirely.

## BetterInfo+ Additions
- <cg>Visibility filter</c> button in level browsers (all / public / unlisted / unlisted-only / friends-only)
- <cg>Page sort</c> button (downloads, likes, newest, oldest — ascending & descending)
- <cg>Completed filter</c> button (hide / only completed levels)
- <cg>Copy level IDs</c> button (copies all IDs on the current page)
- <cg>Visibility badges</c> on level cells (UNLISTED / FRIENDS ONLY)
- <cg>Global sort</c> button in the Filters search view (sorts across *all* loaded results)

The best epic modification ever made for Geometry Dash is here! BetterInfo, now available for Geode!
